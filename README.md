# World Cup Vibe Maker

World Cup Vibe Maker is a fun second screen companion for football fans. It helps users create instant matchday messages, WhatsApp banter, Facebook statuses, X posts, goal reactions, half time verdicts, and full time messages while watching World Cup matches.

## Core Idea

During a big match, fans often use a second screen to chat with friends, react to goals, complain about decisions, celebrate big moments, and post quick thoughts. World Cup Vibe Maker gives them the words quickly.

The user taps the match moment, chooses the mood, selects the platform, and copies a ready to share message.

## Included Features

- Pre Match Banter
- Goal Reaction
- Half Time Verdict
- Full Time Message
- WhatsApp style messages
- Facebook style statuses
- X style short posts
- Funny, passionate, savage, emotional, and balanced tones
- Copy message button
- Generate another button
- Mobile friendly interface

## Technical Approach

This is a static single page app built with:

- HTML
- CSS
- Vanilla JavaScript

It does not use:

- Backend services
- Login
- Database
- Live score data
- Fixture feeds
- API keys
- External AI calls

## How To Run Locally

Open `index.html` in a browser.

## How To Publish With GitHub Pages

1. Go to repository settings.
2. Open Pages.
3. Select the `main` branch.
4. Select the root folder.
5. Save.

The site will be published as a GitHub Pages website.
